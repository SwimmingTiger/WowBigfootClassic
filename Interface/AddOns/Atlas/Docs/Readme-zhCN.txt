==============
= 关于 Atlas =
==============

暴雪为艾泽拉斯世界的每一片区域都绘制了精彩的地图，但出于某种
原因，副本则没有这种地图。当你身处副本时你可以通过微缩地图来
观察周围的区域，但是无法浏览整个副本的概况——这就是 Atlas 
插件用以解决的问题。Atlas 的核心部分基本上是一款图片浏览器，
不过，这些图片使得 Atlas 变得非常有价值——游戏中所有的副本
地图都可浏览。不仅如此，在这些地图上也同时标注出了各个首领的
位置以及其他有用的地点。Atlas 同时包括了战场地图、地下城分布
图、飞行路线图以及户外团队首领分布图。

在正确的安装 Atlas 后，你可以在微缩地图周围发现一个新的图标，
这个图标看起来像是一个球形的一部分（你可以按住鼠标右键对这个
图标在微缩地图周围进行拖动）。单击这个图标就可以打开 Atlas 
的主窗口，里面显示了地图以及位置信息，同时你也可使使用命令
“/atlas”来打开主窗口。在窗口上部的下拉菜单可以帮助你在不同
的地图之间进行切换。你也可以拖动 Atlas 主窗口以移动它的位置，
在右上方的锁形按钮可以使主窗口的位置锁定。

在文字信息下方有一个小型的文本输入框以及“搜索”和“重置”按
钮，在文本框内输入首领名或其它文字并点击“搜索”按钮后，文字
信息将会只显示符合您输入的字符的信息——需要注意的是搜索功能
仅对当前浏览的地图有效，并不是搜索数据库内的全部内容。搜索功
能对于怒焰裂谷这样的小型副本作用甚微，但是对于诸如黑石深渊这
样的副本或 AtlasMajorCities 增强插件而言，该功能将会是获取有
效信息的得力助手。如果你想使整个文本信息复原，只需要点击“重
置”按钮。



========
= 设置 =
========

Atlas 拥有几个简单的设置选项，通过点击主窗口右上部的“设置”
按钮，或者输入命令“/atlas options”即可打开设置窗口。设置窗
口上那些简短的说明不够明确，所以这里特给出有关这些选项的详细
信息。“自动选择副本地图”功能会在你打开 Atlas 的时候侦测你
所在的副本并自动切换到该区域的地图，该功能对于拥有多张地图的
副本无效，诸如黑石塔、厄运之槌以及许多的外域副本，这样的话，
Atlas 将会记住你最后显示的地图。“点击右键打开世界地图”功能
会当你在 Atlas 窗口中进行右键点击后打开世界地图窗口，这个功
能对于快速切换到世界地图浏览方面非常有用。“使 Atlas 不超出
游戏画面”功能会使你对 Atlas 窗口进行拖动时不至于把窗口拖出
游戏界面。其他选项从名字上就能辨别其功能了。如果 Atlas 窗口
被拖动到屏幕外，“重置位置”按钮会非常有用。



==================
= Atlas 增强插件 =
==================

Atlas 有一个内置的插件系统以允许轻松的增加地图包。事实上，
Atlas 本身就有4个这样的增强插件（即战场地图、地下城分布图、
飞行路线图、户外首领分布图）。如果你有意制作你自己的 Atlas 
额外地图包，我建议你先对这4个地图包进行研究。如果你需要帮助
可以主动联系我，也可以在 Atlas 论坛上发帖，不过我仍然要建议
进行尽可能多的研究再进行提问。不仅如此，现在有几个 Atlas 增
强插件已经不仅能提供额外地图了——AtlasLoot 提供了每个首领
的掉落列表，AtlasQuest 则提供了每个副本的相关任务。需要明确
这些插件不是我开发的，它们都有自己的开发者。



==================
= 在进行安装之前 =
==================

* 旧版本的 Atlas 在安装前必须删除。
* 对应旧版本 Atlas 的增强插件（如 AtlasLoot、AtlasQuest 等）
必须删除。



============
= 如何安装 =
============

* 将压缩包解压缩到魔兽世界所在目录。
* 你的解压缩软件必须开启保持目录结构功能。
* 如果你有任何疑惑，请阅读常见问题解答：
* http://www.atlasmod.com/phpBB3/viewtopic.php?t=3
